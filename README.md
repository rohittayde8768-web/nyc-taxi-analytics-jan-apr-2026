🚖 NYC Taxi Analytics | End-to-End Data Analytics Project
An end-to-end Data Analytics project using Python, MySQL, and Power BI to analyze 14+ million NYC Yellow Taxi trips, uncover business insights, and build an interactive dashboard for data-driven decision-making.

📖 Project Overview
The New York City Taxi industry generates millions of trips every month. This project analyzes taxi trip data to understand revenue trends, customer behavior, trip patterns, and operational performance.

Starting from raw data, I cleaned and prepared the dataset using Python, solved real-world business questions using MySQL, and built an interactive Power BI dashboard to present insights visually.

🎯 Business Objective
The main goal of this project is to help taxi operators answer key business questions such as:

Which months and hours generate the highest revenue?
Which payment method contributes the most revenue?
Which pickup locations are the most profitable?
What are the customer travel patterns?
How can taxi operations be improved using data?
