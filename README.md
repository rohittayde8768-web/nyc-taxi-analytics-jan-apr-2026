# 🚖 NYC Yellow Taxi Performance Dashboard

An interactive **Power BI Dashboard** built to analyze **NYC Yellow Taxi Trip Records (January–April 2026)**. This project transforms raw trip data into meaningful business insights using **Power Query, DAX, Data Modeling, and interactive visualizations**.

---

## 📌 Project Objectives

- Analyze overall taxi business performance.
- Track key KPIs such as Trips, Revenue, Tips, Fare, Distance, and Duration.
- Understand revenue and trip patterns.
- Analyze payment methods.
- Identify peak pickup hours.
- Enable interactive filtering for better decision-making.

---

## 📈 Key KPIs

- 🚖 Total Trips: **14.62M**
- 💰 Total Revenue: **$442.78M**
- 💵 Total Tips: **$40.77M**
- 💲 Average Fare: **$21.56**
- 📍 Average Trip Distance: **3.38 Miles**
- ⏱ Average Trip Duration: **17.48 Minutes**

---

## 📊 Dashboard Features

- KPI Cards
- Monthly Revenue Trend
- Trips by Pickup Hour
- Trips by Payment Type
- Revenue by Payment Type
- Interactive Slicers
- Clear All Filters Button

---

## 🛠 Tools & Technologies

- Power BI Desktop

- Data Visualization

---

## 📂 Dataset

**Source:** NYC Taxi & Limousine Commission (TLC) Trip Record Data

---

## 💡 Key Insights

- Generated **$442.78M** revenue from **14.62M** trips.
- Credit Card payments generated the highest revenue.
- Revenue peaked in **March 2026**.
- Evening hours recorded the highest number of pickups.
